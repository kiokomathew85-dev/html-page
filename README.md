# Personal Developer Portfolio

A responsive personal website built to showcase my journey as a junior software developer. This project highlights my core technical skills, recent projects (including the JavaScript Calculator), and provides a direct link to my professional work on GitHub.

##  Features
- **Semantic HTML5 Layout:** Structured with a dedicated Header, About Me, and Projects section for clear navigation.
- **CSS Grid Integration:** Uses a modern Grid layout to elegantly display project cards.
- **Responsive Design:** Optimized for various screen sizes using flexible layouts and modern CSS practices.
- **Professional Branding:** Centered header with a clean, consistent color scheme and direct GitHub integration.

##  Technologies Used
- **HTML5:** Used for the semantic structure and content organization.
- **CSS3:** Applied for aesthetic styling, including Google Fonts, background schemes, and the CSS Grid layout.
- **Git/GitHub:** Employed for version control and project hosting.

## 📂 Project Structure
- `index.html`: The main entry point containing the website's structure.
- `style.css`: The stylesheet containing all custom CSS and Grid layouts.
- `README.md`: Project documentation and setup guide.
